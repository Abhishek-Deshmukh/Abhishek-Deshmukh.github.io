Linear Vector Space

A set of vectors(objects)(a,b,c.....)

Linear vector space 'V'

A vector is denoted as |a⟩,|b⟩...

Rules

  • Commutative law of addition

    |a⟩+|b⟩=|b⟩+|a⟩

  • 1|a⟩=|a⟩

  • |a⟩,α|a⟩

  • Associative λ(μ|a⟩)=(λμ)|a⟩

  • Distributive

  • Null vector 0 meaning |a⟩+0=|a⟩

  • |a⟩+|−a⟩=0

Abitrary constants if not mentioned are to be assumed complex and not real

Span of a vector space

Then span of a set of vectors (a,b,c... s) is

Linear sum of original set

|X⟩=α|a⟩+β|b⟩+........+γ|s⟩

  • if, |X⟩=0=>α|a⟩+β|b⟩+........+γ|s⟩=0

Then one of |a⟩ or |b⟩ or .... can be written as a linear combination of others then these vectors are said to be linearly dependent.

  • if |X⟩≠0=>α|a⟩+β|b⟩+........+γ|s⟩≠0

Means these vectors are linearly independent.

Basis Vectors

If V is an N-Dimensional vector space then set of N. Linearly Independent (|e1⟩,|e2⟩.....|en⟩) from basis vector for V.

In general a Basis set must:-

  • Have as many basis vectors as the number of dimensions(in more formal language, the basis vector must span the space)
  • No basis vector must be described as the sum of others(more formally the basis vectors must be Linearly Independent)

a+bi 's complex conjugate is a−bi

(A∗)ij=(A)∗ij

Hermitian conjugate of a matrix

This is also called as the adjoint of a matrix.

A†=(A∗)T=(AT)∗

Basically the transpose of the complex conjugate of the matrix or the complex conjugate of the transpose of the matrix.

Types of special Square Matrices

Diagonal Matrices

The matrics in which al the non-diagonal matrices are 0.

|A|=A11A22A33..

|A−1|=diag[1A111A221A33...]

For two diagonal matrices AB=BA

Lower and Upper Triangle Matrices

The name tells the basic definition. In the lower triangle matrix has zeros in the upper triangle.

|A|=A11A22A33..

Inverse of a lower triangle matrix is a lower triangle matrix.

Inverse of a upper triangle matrix is a upper triangle matrix.

Symmetric and Anti-Symmetric matrices

A symmetric matrix is when A=AT.

An anti-symmetric or a skew-symmetric matrix is when A=−AT

If A is anti-symmetric then so is its inverse

All square matrices can be written as a sum of a symmetric and an anti-symmetric matrix.

A=12(A+AT)+12(A−AT)

Orthogonal Matrices

AT=A−1

The inverse of a orthogonal matrix is also orthogonal.

|A|=1 always.

Any linear operator that can be represented as an orthogonal matrix when applied to a vector does not change the norm of the vector.

Hermitian and Anti-Hermitian matrix

A Hermitian matrix is a matrix which satisfies A=A†

An anti-Hermitian matrix is a matrix which satisfies A†=−A

All real symmetric matrices are hermitian matrices and all real anti-symmetric matrices are anti-hermitian matrices.

Hermitiananity is invariant against inverse.

All matrices can be written as the sum of a hermitian and an anti-hermitian matrix

A=12(A+A†)+12(A−A†)

Unitary matrix

A unitary matrix is defined as

A†=A−1

Real Orthogonal Matrices are a special case of Unitary Matrices.

Since for a unitary matrix A†A=1, the determinant of unitary matrix has unit modulus.

Normal Matrices

AA†=A†A

A normal matrix is a matrix which commutes with its Hermitian conjugate.
